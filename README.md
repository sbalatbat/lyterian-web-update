# Directory Structure

```
docs/
└── JSDocs/
    ├── fonts/
    ├── scripts/
    └── styles/
source/
└── assets/
    ├── icons/
    ├── images/
    ├── scripts/
    └── styles/
specs/
├── adrs/
├── flows/
└── wireframes/
```

## `docs`

*Codebase documentation.*

| Subdirectory | Contents |
| --- | --- |
| `JSDocs` | Generated JavaScript documentation |
| `JSDocs/fonts` | Fonts used for JSDocs |
| `JSDocs/scripts` | Scripts used for JSDocs |
| `JSDocs/styles` | Styles used for JSDocs |

## `source`

*Project source code. HTML files at the top level and peripherals in subdirectories.*

| Subdirectory | Contents |
| --- | --- |
| `assets` | All HTML peripherals |
| `assets/icons` | Image files for the user interface |
| `assets/images` | Non-icon images |
| `assets/scripts` | JavaScript files |
| `assets/styles` | CSS files |

## `specs`

*Project documentation and architectural records.*

| `adrs` | Architectural Decision Records |
| `flows` | User experience flow charts |
| `wireframes` | High-fidelity UI designs for implementation |
